i_task_j

i - позиция задания (сначала рекомендуется выполнять задания с индексом 0,
поскольку для их выполнения достаточно знаний предыдущего семестра, потом
задания с индексом 1 и т.д.)
task - название задания
j - максимальное количество баллов за задание при выполнении всех условий,
при написании чистого качественного кода самостоятельно

Выполнив все задания можно получить 31.75 баллов
(без учёта OneChanceToChoose и с учётом одной сданной игрой)
если учесть всё, то можно получить 35+ баллов.
По предварительным расчётам для пятёрки достаточно 16 баллов с учётом контрольных.
